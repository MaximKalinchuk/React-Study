Инфо

Суть данного сайта в том, что на нём будут размещены общие задачи по React, с различных курсов. Уточню, тут не будет готового решения, только визуальное представление и описание задач. Во избежания претензий на плагиат некоторые задачи видоизменены. Все задачи выполнены исключительно на React. Сделано это для демонстрации навыков работодателю и чтобы данные таски не были утеряны со временем. Если у вас есть свободное время, вы можете пробовать реализовать их самостоятельно.

Задание 1 - Форма регистрации

После отправки формы появляется таблица, в которой показываются значения всех полей. Из этой формы можно вернуться в редактирование по кнопке Back. При этом все данные оказаются на своих местах.


Задание 2 - Плюс / Минус

Нажмите на +, чтобы добавить новую кнопка в верх списка. На -, чтобы удалить кнопку сперху списка. Также при нажании на сами кнопки, они удаляется.
