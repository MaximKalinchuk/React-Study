О проекте

Суть данного сайта в том, что на нём будут размещены общие задачи по React, которые я решал. Уточню, тут не будет готового решения, только визуальное представление и описание задач. Во избежание претензий на плагиат некоторые задачи изменены. Все задачи выполнены исключительно на React. Сделано это для демонстрации навыков работодателю. Если у вас есть свободное время, вы можете пробовать реализовать их самостоятельно.

Задание - Форма регистрации

После отправки формы появляется таблица, в которой показываются значения всех полей. Из этой формы можно вернуться в редактирование. При этом все данные оказываются на своих местах.


Задание - Плюс / Минус

Нажмите на +, чтобы добавить новую кнопка вверх списка. На -, чтобы добавить кнопку с числом на 1 меньше предыдущего. Также при нажатии на сами кнопки, они удаляются.


Задание - ToDo лист

Формирует список ваших тасков. После нажатия на зеленый минусик, таск переходит в раздел выполненных тасков. Если нажать на минусик в выполненных тасках, задача удаляется. Состояние сохраняется в LocalStorage.
